Caso de Uso: Calcular VAI (Índice de Adiposidad Visceral)
ID: CU-03
Actor principal: Médico
Ámbito: Calculadora de Salud
Nivel: Objetivo de Usuario
Descripción: El médico quiere calcular el Índice de Adiposidad Visceral (VAI) de un paciente. El sistema debe validar que los datos (sexo, IMC, circunferencia de cintura, triglicéridos y colesterol HDL) tengan sentido físico y realizar el cálculo de forma adecuada según el sexo del paciente.
Interesados:
- Médico: Quiere conocer el VAI del paciente para poder orientarle sobre su salud y riesgo cardiometabólico.
- Paciente: Conocer su índice de adiposidad para comprender su riesgo cardiovascular.

Precondiciones:
- El sistema (HealthCalc) debe estar activo.
- El médico debe tener acceso al sistema.

Disparador: El médico solicita Calcular VAI.
Garantía de éxito: El sistema devuelve el VAI correcto del paciente.
Garantía mínima: El sistema devuelve un mensaje de error.

Escenario principal de éxito:
1. El médico introduce el sexo, IMC, circunferencia de la cintura (CC), triglicéridos (TG) y colesterol HDL del paciente en el sistema.
2. El sistema valida que los datos introducidos tienen sentido físico y biológico.
3. El sistema calcula el VAI utilizando la fórmula matemática correspondiente según el sexo introducido.
4. El sistema muestra el valor VAI correcto del paciente al médico.

Extensiones:
2a. Sexo inválido: El sistema muestra un mensaje indicando que el sexo debe ser 'm' (masculino) o 'f' (femenino) y solicita un nuevo valor. Vuelta al paso 1.
2b. Parámetros inválidos o fuera de rango: El sistema muestra un mensaje que indica que uno de los datos (IMC, CC, TG o HDL) no es válido o escapa de los límites humanos realistas, y solicita un nuevo valor. Vuelta al paso 1.
